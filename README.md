# 실시간 멀티플레이어 야구 게임 (완전판)

Socket.IO 기반의 실시간 네트워크 동기화가 적용된 멀티플레이어 야구 미니게임입니다. 

## 구현된 주요 기능
* **실시간 매치메이킹**: 서버 내부 큐를 통한 1vs1 자동 매칭 시스템.
* **지연 시간(Ping) 보정**: 클라이언트와 서버 간 왕복 지연 시간을 계산하여 공의 비행 궤적 및 타격 스윙 타이밍을 정밀 보정합니다.
* **물리 궤적 애니메이션**: 직구(강속구 형태 빠른 이동) 및 커브(사인파 곡선 탄도 이동) 구현.
* **이닝 및 공수교대 규칙**: 3스트라이크 아웃, 4볼 진루, 3아웃 시 투수와 타자 역할 자동 전환 스왑 메커니즘 제공.

!플레이어 1,2의 인터넷을 열어 `http://localhost:3000`에 각각 접속한 뒤 매칭을 진행해 주세요!
